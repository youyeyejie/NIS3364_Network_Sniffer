\chapter{项目基本信息}


\section{项目简介}
本项目是一个基于Python Scapy库开发的网络嗅探器工具，能够实时捕获和分析网络数据包，支持多种协议解析、数据包过滤、混杂模式和分片重组等功能。该工具提供了直观的图形界面，便于网络分析和故障排查。项目已发布在 \href{https://github.com/youyeyejie/NIS3364_Network_Sniffer}{Github-NIS3364 Network Sniffer}.

\section{开发环境}
\begin{itemize}[label=$\bullet$]
    \item Python 3.10 或更高版本
    \item 依赖库：
    \begin{itemize}[label=$\bullet$]
        \item scapy >= 2.4.5
        \item psutil >= 5.9.0
        \item tkinter (Python标准库)
    \end{itemize}
\end{itemize}
